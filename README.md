# LMS Project — MERN Education Platform

A mid-level full-stack project built with MongoDB, Express, React, and Node.js.

## Features
- JWT Authentication (Login / Register)
- Role-based access (Student & Instructor)
- Instructor can create, update, delete courses
- Students can enroll in courses and track progress
- Protected routes on frontend and backend

## Tech Stack
- **Frontend:** React.js, React Router, Axios, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas + Mongoose
- **Auth:** JWT + bcrypt

## Setup
1. Clone the repo
2. Backend: `cd backend && npm install && npm run dev`
3. Frontend: `cd frontend && npm install && npm run dev`
4. `.env` in `MONGO_URI` and `JWT_SECRET` start
